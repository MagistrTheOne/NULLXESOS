# NULLXES OS 0.1.0

First public alpha. Establishes the Stage 1–2 desktop layer on top of vanilla
Arch Linux.

## Highlights

- **FRAME compositor** (smithay 0.6, hybrid in-process panel).
- **External clients**: LAUNCHER, SLATE, NX-LOCK, NX-NOTIF, NX-SETTINGS, NX-GREET.
- **Control IPC v1** for tooling.
- **Theme pipeline** generates GTK 3/4, Qt 6, Plymouth, systemd-boot, and icon-theme assets from a single TOML.
- **Live ISO** with Calamares; btrfs + snapper + systemd-boot.
- Pacman repo signed with hardware-backed key.

## Known limitations

- Multi-monitor heterogeneous DPI / fractional scaling: deferred to 0.2.
- Server-side decorations: deferred to 0.2.
- NVIDIA proprietary: experimental.

## Upgrade

This is the first release; clean install only.
